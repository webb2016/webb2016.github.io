如何创建GitHub个人博客

  • GitHub
  • Hexo
技术

想不想拥有个人博客，那么按照本文教程自己动手吧！

1、环境搭建
1.1安装Node.js 官网：https://nodejs.org/
1.2安装Git 官网：http://git-scm.com/
1.3安装Hexo
打开Git Bash

$ npm install -g hexo-cli

在电脑中建立一个名字叫Hexo的文件夹，然后在此文件夹中右键打开Git Bash。执行下面的命令

$ hexo init nodejs-hexo

在Hexo3中,hexo-server独立出来了,如要本地调试,需要先安装

$ npm install hexo-server --save

到\node_modules目录下面看自己是不是已经安装了这些模块,如果没有则安装

npm install hexo-generator-index --save
npm install hexo-generator-archive --save
npm install hexo-generator-category --save
npm install hexo-generator-tag --save
npm install hexo-server --save
npm install hexo-deployer-git --save
npm install hexo-renderer-marked@0.2 --save
npm install hexo-renderer-stylus@0.2 --save
npm install hexo-generator-feed@1 --save
npm install hexo-generator-sitemap@1 --save

启动本地服务

$ hexo generate
$ hexo server

在浏览器地址栏输入local:4000即可看到Hexo页面

2、注册GitHub
官网：http://www.github.com/

3、配置GitHub
3.1建立Repository

仓库名必须为【your_user_name.github.io】

3.2配置SSH keys
可以用Git Gui的ssh工具生成key

添加SSH Key到GitHub
登陆github系统，点击右上角的 Account Settings—->SSH Public keys —-> add another public keys
把你本地生成的密钥复制到里面（key文本框中），点击add key就ok了

4、部署Hexo
GitHub博客已经建立好了，那如何将本地的Hexo页面上传到我们的博客上呢？
编辑 _config.yml (在Hexo文件下)，把账号名换成你自己的即可，注意这里冒号有空格

deploy:
  type: git
  repository: git@github.com:webb2016/webb2016.github.io.git
  branch: master

执行下列指令即可完成部署。

hexo generate
hexo deploy

这时打开你的博客，如我的是http://webb2016.github.io/
即可看到之前在本地看到的Hexo页面

5、Hexo高级设置
5.1安装主题
GitHub上有很多很棒的主题，如果想要我这个主题，点击我

将Git Shell 切到Hexo目录下，然后执行下面的命令，将主题下载到 themes目录下。

$ git clone https://github.com/litten/hexo-theme-yilia.git

修改博客根目录Hexo下的_config.yml配置文件中的theme属性，将其设置为themes目录下的主题文件名。
theme: hexo-theme-yilia
5.2添加评论区
我们下载的主题里面包含了第三方评论系统，到主题目录里面的_config.yml中去查看，如使用多说评论系统，
则到多说官网去注册，在_config.yml将得到的duoshuo-key输入
这样的话可能会出现很多莫名其妙的评论，解决方法如下：
假设在多说网站的二级域名为xxxx.duoshuo.com
我们需要打开themes/hexo-theme-yilia/layout/_partial/post/duoshuo.ejs
将var duoshuoQuery = {short_name:”<%=theme.duoshuo%>”};修改为var duoshuoQuery = {short_name:”xxxx”};。

在Hexo/source目录下创建自己的 404.html，将下面的代码拷入其中

---

    <head>
         <meta charset="UTF-8" />
         <title>404</title>                                                                                                                                        
    </head>
    <body>
        <script type="text/javascript" src="http://www.qq.com/404/search_children.js" 
        charset="utf-8" homePageUrl="http://webb2016.github.io/" homePageName="回到我的主页"></script>
    </body>
</html>
